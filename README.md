# Electron Boilerplate

## What's included out-of-the-box?

- Electron Forge
- Vite
- React
- TailwindCSS
- shadcn/ui
- Full Typescript support
- React-router-dom HashRouter

## How to get started?

1. Install dependencies

```bash
yarn
```

2. Start the project

```bash
yarn start
```
